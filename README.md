# Radare Skills

This repository contains useful skills to help your agents use radare2 better

* Code Review
* Development and Contribute
* Finding Vulnerabilities
* Explotiing Bugs
* Analizing Binaries
* Create function signatures
* Comparing binaries
* Improve decompiler output
* Forensics on unknown file formats
* Using r2frida, r2yara, etc
* ...

Feel free to contribute with your own skills, or update the existing ones.

This repo will be useful not just for agents, but also to users as a central point of documentation.
